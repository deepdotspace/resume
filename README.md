# Resume Builder

A LaTeX-based resume and CV builder built on the [DeepSpace SDK](https://github.com/deepdotspace).

**Live**: [resume-builder.app.space](https://resume-builder.app.space)

## Features

- Reusable profiles — shared career data across multiple resumes
- Multiple LaTeX templates — Modern, Europass, Academic CV, Two-Column, Jake's Resume
- Live PDF preview with cloud compilation (pdfLaTeX, XeLaTeX, LuaLaTeX)
- Version history with downloadable PDF snapshots
- Job-description tailoring — rephrase per-section against a target role
- One-shot AI assists — improve bullets, rewrite summaries, suggest skills
- In-editor AI chat assistant — edits the resume directly (form mode) or
  the raw LaTeX (override mode)
- Resume upload (PDF/DOCX) with AI-driven field extraction
- Glassmorphic UI with user-selectable backgrounds
